### Logrus-kit is a kit for logrus.
**Logrus-kit inspired by [Logrus mate](https://github.com/gogap/logrus_mate)** 

#### Install
```
go get github.com/Kamva/logrus-sdk
``` 

#### Formatter list:
- json
- text

#### Hook list:
- slack

#### Output list
- stdout
- stderr
- null
